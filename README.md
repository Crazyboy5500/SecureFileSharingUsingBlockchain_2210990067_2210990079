## 🔍 About the Project

A decentralized file sharing application built with React, IPFS (Pinata),
and Ethereum smart contracts deployed on the Sepolia Testnet.

---

## ✨ Features

- 🔐 MetaMask Wallet Authentication
- ⏱️ 30-Minute Session Management with auto logout
- ☁️ IPFS Upload via Pinata with progress tracking
- ⛓️ Blockchain Storage on Ethereum Sepolia
- 🔍 File Verification using SHA256 hashing
- 📂 Paginated File Browser
- 📱 Fully Responsive Design

---

```bash

# Go into source code folder
cd Source_Code

# Install dependencies

# Start the app
npm start
```
